# Flowers-Shop

Ce projet est un gestionnaire de tâches collaboratif qui permet à l'admin d'assigner des tâches aux employés.
Les employés peuvent consulter leurs tâches et les valider.

Ce projet a été fait dans le cadre de la formation Développeur Web et web mobiles au CEPPIC d'Isneauville - 2025
