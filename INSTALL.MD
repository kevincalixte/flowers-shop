# Installation

WAMP : https://www.wampserver.com/
URL : http://localhost/flowers-shop
BDD : user = root / pass = "" [voir config/database.php]

- Télécharger le projet
- Dézipper dans le dossier www dans -> C:\wamp64
- Lancer WAMP
- Importer la BDD en executant database.sql dans PHP My Admin (Si besoin, l'utilisateur est "root" et mot de passe "")
- Ouvrir la page sur localhost ou 127.0.0.1
- Entrer un email utilisateur et un mot de passe pour vous connecter : lucas.petit@flowers-shop.com (Responsable) ou sophie.leroy@flowers-shop.com (Vendeur)
